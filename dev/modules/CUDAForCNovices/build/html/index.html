CUDA For C Novices¶

This module is for students learning CUDA who are new to C. It introduces the basics you will need to know through a lab activity and some follow-up questions. Here are the two code samples you will need (they are also written out in this module).
  • addVectors.cu
  • divergence.cu

This lab was created by Professor David Bunde of Knox College.
